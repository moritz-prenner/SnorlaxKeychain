# SnorlaxKeychain
A blinking and vibrating Snorlax made out of a PCB.
## Parts List
6x 5mm LED
6x 220 Ohm resistor
1x 4.7k Ohm resistor
1x 2N3904 transistor (NPN)
1x mini motor disc
1x 6mm push button
1x CR2032 battery cell
